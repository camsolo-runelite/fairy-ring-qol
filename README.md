# Fairy Ring QoL
Faster Fairy Ring selection with keyboard shortcuts and clipboard support.

## Features

### ⌨️ Select With Keyboard
Press any hotkey (default = Enter) to select the first Fairy Ring in the list.  
*Note: You may need to wait a tick for search results to render first.*

### ⚡ Auto-Select
Automatically select Fairy Ring when a valid code is typed.

### 📋 Clipboard Support
Press **Ctrl+V** while the fairy ring interface is open to paste a code directly from your clipboard.
